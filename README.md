# MidnightCheck — WoW Midnight Addon

A dark-themed raid readiness tracker. Scans your character for missing consumables, enchants, and gems before you step into a raid or Mythic+ key.

## Features

- Consumable checks — food, flask, weapon oil, potions, health potions
- Enchant scanner — flags any gear slot missing a required enchant for your spec
- Gem scanner — counts empty sockets
- Quick-use buttons — left-click to use, right-click to swap tracked item
- Auto-scan on raid entry
- Minimap button

## Usage

```
/mc      Run a readiness scan
/midnight  Alias for /mc
```
